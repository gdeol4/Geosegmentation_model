# Creating a Geosegmentation model to examine bank churn

A geosegmentation model examining the churn of bank customers using regression.

## The Columns ##

* **RowNumber**
* **CustomerID**
* **Surname
* **CreditScore
* **Geography
    + **Germany
    + **France
    + **Spain
* **Gender
    + **Male
    + **Female
* **Age
* **Tenure
* **Balance
* **NumOfProducts
* **HasCrCard
* **IsActiveMember
* **EstimatedSalary
* **Female
